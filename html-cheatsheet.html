HTML Cheat Sheet

Key HTML elements, attributes, and functionalities:

  • <!DOCTYPE html> – Declares the document as HTML5.
  • <html> – Root element of the document.
  • <head> – Contains meta-information and links.
  • <meta> – Sets metadata like character set.
  • <title> – Defines the document title.
  • <link> – Links external files such as CSS.
  • <body> – Contains the visible page content.
  • <header> – Represents introductory content.
  • <nav> – Defines navigation links.
  • <section> – Represents a section of content.
  • <article> – Specifies independent content.
  • <aside> – Marks side content.
  • <footer> – Contains footer information.
  • <h1>-<h6> – Headings of different levels.
  • <p> – Defines a paragraph.
  • <a> – Creates hyperlinks.
