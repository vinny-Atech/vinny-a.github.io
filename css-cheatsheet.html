CSS Cheat Sheet

Key CSS properties, selectors, and functions:

  • color – Sets text color.
  • background-color – Sets background color.
  • font-size – Adjusts text size.
  • font-family – Specifies fonts.
  • margin – Controls outer spacing.
  • padding – Controls inner spacing.
  • border – Defines element borders.
  • display – Sets the display type (block, inline, flex).
  • position – Specifies element positioning.
  • z-index – Controls stacking order.
  • flex – Enables flexbox layout.
  • grid – Enables grid layout.
  • transition – Animates property changes.
  • transform – Applies 2D or 3D transforms.
  • @media – Adds responsive styles.
